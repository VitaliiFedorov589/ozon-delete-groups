# ozon-delete-groups

Браузерный скрипт для массового удаления групп связанных товаров в Ozon Seller.

> Пошаговая инструкция для пользователей без опыта в программировании — в файле [ИНСТРУКЦИЯ.md](ИНСТРУКЦИЯ.md).

## Как работает

Скрипт запускается в консоли браузера на странице [Группы связанных товаров](https://seller.ozon.ru/app/products-merge?tab=related_products).

1. Находит кнопки удаления групп через DOM-запрос по уникальным координатам SVG-иконки корзины (`6.333`, `5.834`, `8.333`).
2. Кликает кнопку удаления, затем кнопку подтверждения `Удалить` в модальном окне.
3. Повторяет до достижения заданного количества или пока группы не закончатся.

### Паузы (защита от блокировки)

| Условие | Пауза |
|---|---|
| Между каждым удалением | 300–600 мс |
| После клика по кнопке удаления | 200–400 мс |
| Каждые 20 удалений | 3–5 сек |
| Каждые 500 удалений | 5 минут |

Если кнопок удаления не видно — скрипт прокручивает страницу вниз и повторяет поиск.

## Параметры

**Количество удаляемых групп** — последняя строка скрипта:

```js
deleteGroups(3301); // заменить 3301 на нужное число
```

## Остановка

Ввести в консоли браузера и нажать Enter:

```js
window.stopDeletion = true;
```

Скрипт завершится после текущей итерации и выведет количество удалённых групп.

## Требования

- Любой современный браузер (Chrome, Edge, Firefox)
- Авторизованный аккаунт продавца на [seller.ozon.ru](https://seller.ozon.ru)
- Доступ к DevTools (клавиша F12)

## Предупреждение

Удаление групп **необратимо**. Убедитесь, что вы находитесь на правильной странице и понимаете, что делаете.
